# Expense Tracker

## Project Overview

Expense Tracker is a modern full-stack web application that helps users manage their daily expenses efficiently. Users can add, update, delete, and monitor their expense records in a clean and responsive interface.

This application makes expense management simple by allowing users to keep track of spending habits and organize expenses by category and date.

---

## Features

- **Add Expense**
  Users can add new expense records with title, amount, category, and date.

- **View Expense List**
  Display all expenses in a structured table format for easy tracking.

- **Update Expense**
  Edit existing expense information anytime.

- **Delete Expense**
  Remove unnecessary expense records permanently.

- **Category-Based Tracking**
  Organize expenses by categories such as Food, Shopping, Transport, Bills, Health, etc.

- **Responsive Design**
  Fully responsive UI for desktop, tablet, and mobile devices.
